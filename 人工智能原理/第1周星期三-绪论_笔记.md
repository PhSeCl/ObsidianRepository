# 第1章 绪论 —— 什么是人工智能

> [!info] 本章导览

> 📌 本章原始课件以大量产品演示、影视片段、新闻截图为主（无教学示意图），故下文核心框架均用**自绘表格 / ASCII 图**呈现。

## 1.1 什么是人工智能（Artificial Intelligence, AI）？

历史上对 AI 的定义可按两个维度划分：**「思考 vs 行动」** × **「像人 vs 理性」**，形成四象限。

| | **关注「思考」（内部过程）** | **关注「行动」（外部表现）** |
|---|---|---|
| **像人（Human-like）** | **类人思考**：认知建模方法<br>→ 认知科学，用真实人/动物实验验证模型 | **类人行动**：图灵测试方法<br>→ 让机器表现得与人无法区分 |
| **理性（Rational）** | **理性思考**：「思维法则」方法<br>→ 逻辑学，正确前提必得正确结论 | **理性行动**：**理性智能体**方法 ✅<br>→ 采取行动以取得最佳（期望）结果 |

> [!important] 本课程的立场：理性行动（Acting Rationally）
> **理性（Rational）= 最大化你的期望效用（maximizing expected utility）**。
> - 理性只关心**做什么决策**，不关心**决策背后的思考过程**。
> - 目标用**结果的效用（utility of outcomes）**来表达。
> - 在**不确定性**存在时，追求最佳**期望**结果。
>
> 老师笑称：这门课更贴切的名字其实是 **「计算理性（Computational Rationality）」**。

> [!example] 图灵测试需要机器具备的六大能力
> 要在「类人行动」测试中蒙混过关，机器至少需要：
> 1. 自然语言处理（natural language processing）
> 2. 知识表示（knowledge representation）
> 3. 自动推理（automated reasoning）
> 4. 机器学习（machine learning）
> 5. 计算机视觉（computer vision）—— 完全图灵测试还需此项
> 6. 机器人学（robotics）—— 完全图灵测试还需此项

### 智能体（Agent）与环境的交互循环

本课程的统一视角：**智能体 = 感知（percept）并行动（action）的实体**。理性智能体选择能最大化其期望效用的行动。

```text
        ┌─────────────────── 智能体 Agent ───────────────────┐
        │                                                     │
        │   传感器 Sensors  ──► [ 决策/推理 ] ──►  执行器       │
        │      ▲ Percepts                          Actuators  │
        └──────┼──────────────────────────────────────┼──────┘
               │                                       │
               │ 感知                              行动 │
               │                                       ▼
        ┌──────┴───────────────────────────────────────────┐
        │                    环境 Environment                │
        └───────────────────────────────────────────────────┘
```

> [!note] 决定「如何理性行动」的三要素
> **感知（percepts）**、**环境（environment）**、**行动空间（action space）** 的特征，共同决定了选择理性行动所需的技术。本课程正是围绕「不同环境特征下该用什么技术」展开。

## 1.2 人工智能的基础（贡献学科）

AI 不是凭空出现的，它站在多门学科的肩膀上：

| 学科 | 对 AI 的贡献（举例） |
|---|---|
| 哲学 | 亚里士多德「三段论（Barbara syllogism）」推理 |
| 数学 | 概率论、逻辑、计算 |
| 经济学 | 决策论、博弈论、效用 |
| 神经科学 | 神经元模型 |
| 心理学 | 人机交互、认知、智能增强 |
| 计算机工程 | 摩尔定律、算力 |
| 控制论 | 最小化代价函数（反馈控制） |
| 语言学 | 自然语言处理 |

> [!example] 亚里士多德的 "Barbara" 三段论（逻辑推理的雏形）
> - 大前提：所有人（M）都会死（D）：$\forall x\,[M(x) \Rightarrow D(x)]$
> - 小前提：所有希腊人（G）都是人（M）：$\forall x\,[G(x) \Rightarrow M(x)]$
> - 结论：所有希腊人（G）都会死（D）：$\forall x\,[G(x) \Rightarrow D(x)]$

## 1.3 人工智能的（简短）历史

```text
1943 ─ McCulloch & Pitts：神经元的「开/关」逻辑模型
1950 ─ Turing《计算机器与智能》：提出图灵测试、机器学习、强化学习等设想
1956 ─ 达特茅斯会议（Dartmouth）：正式提出 "Artificial Intelligence" 一词
1965 ─ Robinson：完整的一阶逻辑定理证明算法
1969-79 ─ 专家系统 MYCIN（450条规则，诊断血液感染，优于初级医生）
1988-93 ─ 「AI 寒冬」：研究受挫、预算衰减
1990-2010 ─ 统计方法 + 不确定性推理兴起
2006 ─ Hinton 深度信念网络快速学习算法（深度学习复兴）
2012 ─ AlexNet 赢得 ImageNet（GPU + 大数据 + 深度学习）
2016 ─ AlphaGo 击败李世石
2017 ─ Transformer 架构问世；AlphaFold 预测蛋白质结构
2022 ─ OpenAI 发布 ChatGPT，进入大语言模型（LLM）时代
```

> [!summary] 历史脉络的三次范式
> 1. **符号主义 / 基于知识**（早期—1980s）：逻辑、专家系统。
> 2. **统计学习**（1990s—2010s）：概率、不确定性推理。
> 3. **深度学习 / 大模型**（2012—至今）：大数据 + 算力 + 神经网络。
> 理解「为什么从符号主义演化到统计学习再到深度学习」，比记住时间点更重要。

## 1.4 目前的先进技术

- **自然语言处理**：语音识别（ASR）、语音合成（TTS）、问答、机器翻译、网络搜索、文本分类、大语言模型。
- **计算机视觉**：图像描述（captioning）、目标检测、图像分割、Deep-Fake 换脸。
- **机器人学**：工业机械臂、人形机器人、自动驾驶汽车（如 Waymo）。

## 1.5 人工智能的风险与收益

> [!warning] 风险
> - 致命性自主武器
> - 监视与劝诱（如社媒推荐操纵）
> - 有偏决策（如贷款审批中的种族歧视）
> - 就业影响（如电话客服被替代）
> - 安全关键应用的失效（如自动驾驶事故）
> - 网络安全（如 AI 生成钓鱼/伪造录音）

> [!note] 收益
> 辅助残障人士、医学影像辅助诊断、AI 导师、AI 养老监护、地震预测等。
> 核心观点：**当前大模型应作为「辅助工具」供人参考校验，而非「替代」人**（如医疗诊断仍需医生把关）。

## 本章小结

> [!summary] 关键要点
> 1. AI 的四种定义视角中，本课程采用 **「理性行动 / 理性智能体」**。
> 2. **理性 = 最大化期望效用**，只关心决策结果而非思考过程。
> 3. 智能体通过 **感知—决策—行动** 与环境交互；环境特征决定所需技术。
> 4. AI 历史经历 **符号主义 → 统计学习 → 深度学习/大模型** 三次范式。
> 5. 全课程围绕 **搜索/逻辑、概率推理、监督学习、强化学习** 四大板块。
