# 人工智能原理 — 课程总览（MOC）

> [!abstract] 关于本知识库
> 本 Vault 由中山大学《人工智能原理》（彭振辉，2026 春）全部 29 个课件重构而成，覆盖**搜索、逻辑、概率推理、决策与学习**五大板块。所有笔记图文并茂、含 Callout、LaTeX 公式、双链与自测题。教材：《人工智能 — 一种现代的方法（第 4 版）》。

## 学习路线图

```
绪论(理性智能体)
   │
   ├── 搜索板块 ── 智能体 → 无信息搜索 → 启发式A* → 局部搜索 → CSP → 对抗搜索 → MCTS
   │
   ├── 逻辑板块 ── 命题逻辑 → 推理(模型检验/前向链接/DPLL) → 一阶逻辑
   │
   ├── 决策板块 ── MDP(价值/策略迭代) → 老虎机 → 强化学习(TD/Q-learning) → 简单决策(效用/VPI)
   │
   ├── 概率板块 ── 贝叶斯规则 → 贝叶斯网络 → 变量消元 → 采样 → 马尔可夫模型(HMM/滤波/粒子滤波)
   │
   └── 学习板块 ── 监督学习 → 决策树 → 感知器 → 神经网络 → 反向传播 → 逻辑回归
```
